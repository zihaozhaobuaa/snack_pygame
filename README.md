### 游戏说明

#### 玩法

- 按任意方向键开始游戏
- 通过上下左右键控制蛇的移动
- 吃到食物使蛇身变长
- 游戏中有越界、吃到自己、蛇身长度达到一定条件（50格）游戏结束的判断
- 按q退出游戏，按c重新开始
